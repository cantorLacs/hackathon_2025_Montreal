🔧 Opción 2: Solucionar WebGL

Para usar Three.js 3D, necesitas WebGL funcionando:

📋 Pasos para Habilitar WebGL:

  • 🟦 Chrome/Edge:
    1. Ir a chrome://settings/
    2. Buscar "Configuración avanzada"
    3. En "Sistema" activar "Usar aceleración por hardware"
    4. Reiniciar navegador
  • 🟧 Firefox:
    1. Ir a about:config
    2. Buscar webgl.disabled
    3. Cambiar a false
    4. Reiniciar navegador
  • 🟪 Safari:
    1. Preferencias → Avanzadas
    2. Activar "Mostrar menú Desarrollador"
    3. Desarrollador → Activar WebGL
    4. Reiniciar navegador
